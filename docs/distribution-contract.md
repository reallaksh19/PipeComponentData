# Distribution Contract

This repo is the shared pipe component data and adapter package.

Target consumers:

- 3D Viewer via UXML adapters.
- GLB-PCF-2.5-Editor via CEG adapters.
- Simplified Analysis via canonical geometry adapters.
- Future converter apps via package or pinned ESM URL.

Phase 0/1 exposes browser-safe and Node-safe ESM. Later releases may publish tagged builds for import maps and npm/git dependencies.
